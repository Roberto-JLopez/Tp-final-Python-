## Descripción del Sistema
Simulador de caja de supermercado ejecutado por consola. El sistema gestiona el flujo completo de ventas: carga de productos mediante código, cálculo de totales, aplicación de descuentos promocionales, generación de tickets y registro en historial continuo (`ventas.txt`). Implementa validaciones robustas, manejo de excepciones y ordenamiento de estadísticas diarias.

## Instrucciones de Ejecución
1. Clonar este repositorio.
2. Verificar que `productos.txt` y `ventas.txt` estén ubicados exactamente en el mismo directorio que el script principal.
3. Ejecutar en la terminal el siguiente comando:
   `python supermercado.py`
